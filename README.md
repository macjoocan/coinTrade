## 2. 핵심 전략 로직

### 2.1 진입 점수 시스템

**12점 만점 스코어링 시스템**

```markdown
## 진입 점수 시스템

진입 점수 = Σ(각 지표 점수)

### 구성 요소:
- **추세 조건**: 최대 3점
- **RSI 지표**: 최대 3점  
- **MACD 신호**: 최대 2점
- **거래량**: 최대 2점
- **변동성**: 최대 2점
```

### 2.2 진입 조건

python

```markdown
```python
if 진입_점수 >= 임계값:
    if 리스크_체크_통과:
        if 시장_상황_적절:
            포지션_진입()
```
```

**임계값 설정**

- 기본: 6점
- 베어장: 7점 (강화)
- 불장: 5.5점 (완화)

### 2.3 청산 조건

**우선순위별 청산 트리거**

1. **손절 (최우선)**: 보유시간 무시, 즉시 실행

3. **목표 수익**: 1.5% 도달 시
4. **시간 기반**: 24시간 초과 보유 시

## 3. 리스크 관리 시스템

### 3.1 Kelly Criterion 적용

```markdown
f* = (p × b - q) / b

f* = 최적 베팅 비율
p = 승률
b = 손익비
q = 1 - p
```

**보수적 적용**: `실제 사용 = f* × 0.25`

### 3.2 포지션 크기 계산

```markdown
기본_크기 = 잔고 × 최대_포지션_비율(20%)

조정_요소:
- Kelly Criterion 배수
- 시장 상황 배수 (0.7 ~ 1.1)
- 변동성 조정 (역비례)
- 연속 손실 페널티

```

### 3.3 손실 제한

| **제한 유형** | **값** | **동작** |
| --- | --- | --- |
| 포지션당 손절 | 1.2% | 즉시 청산 |
| 일일 최대 손실 | 2% | 거래 중단 |
| 연속 손실 | 3회 | 거래 일시정지 |
| 자본 손실 | 5% | 보호 모드 |

## 6. 시장 상황 분석

### 6.1 시장 판단 로직

```markdown
def analyze_market():
    btc_3d_change = get_btc_change(3)
    
    if btc_3d_change > 3:
        return 'bullish'
    elif btc_3d_change < -3:
        return 'bearish'
    else:
        return 'neutral'

### 6.2 상황별 전략 조정
